📊 **LeetMetricApp**

LeetMetric is a responsive web application that allows users to track and analyze their LeetCode problem-solving progress in real time. The application fetches user statistics using the LeetCode API and displays performance metrics in a clean and user-friendly interface.

### 🚀 Features

* Search LeetCode profile by username
* Displays solved problems statistics and progress data
* Real-time data fetching using API integration
* Visual representation of coding progress
* Error handling for invalid usernames
* Clean and responsive user interface
* Dynamic data update without page reload

### 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML & CSS
* REST API

### 📌 Functionality

Users can enter their LeetCode username in the input field, and the application fetches the latest coding statistics from the API. The app processes the response and displays problem-solving data such as solved problems and progress metrics in an organized format. Proper validation and error handling ensure a smooth user experience.

### 🎯 Purpose

This project was developed to practice API integration, React state management, and frontend UI development while building a practical tool for tracking coding performance and improving problem-solving consistency.
